Login and Sharing
=================

You can use PyGraphistry as a client to an optional Graphistry server for benefits such as GPU visual analytics, securely share visualizations, simplify your graph pipelines, and share GPU resources. You can self-host Graphistry on your own GPU instances, or get an account on `Graphistry Hub <https://www.graphistry.com/get-started>`_, including the free GPU tier.
